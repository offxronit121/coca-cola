# 🥤 Coca-Cola: The Future of Refreshment

An award-winning, immersive cinematic experience showcasing the futuristic identity of Coca-Cola. This project is a perfect blend of interactive 3D storytelling and modern web technologies[cite: 1].

---

## ✨ Features

*   **Interactive 3D Bottle**: A high-fidelity iconic glass bottle built with React Three Fiber and Three.js that rotates and pulses in real-time[cite: 1].
*   **Immersive Opening Scene**: A unique "Tap to Open" interaction featuring custom spatial audio (`pop.mp3`) and fizzy confetti effects[cite: 1].
*   **Cinematic Scrolling**: Buttery smooth scroll transitions powered by GSAP and Lenis integration[cite: 1].
*   **Modern UI/UX**: A premium dark-themed visual experience utilizing Tailwind CSS 4 and advanced glassmorphism effects[cite: 1].
*   **Responsive Design**: Fully optimized layout for both high-end desktop displays and mobile devices[cite: 1].

---

## 🛠️ Tech Stack

*   **Frontend**: React 19, TypeScript[cite: 1]
*   **3D Engine**: Three.js, @react-three/fiber, @react-three/drei[cite: 1]
*   **Animations**: GSAP, Motion (formerly Framer Motion)[cite: 1]
*   **Styling**: Tailwind CSS 4[cite: 1]
*   **Smooth Scroll**: Lenis[cite: 1]

---

## 🚀 Getting Started

### Prerequisites
*   Node.js (LTS version)
*   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/your-username/coca-cola-future.git](https://github.com/your-username/coca-cola-future.git)
    cd coca-cola-future
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    The project will be live at `http://localhost:3000`![cite: 1]
